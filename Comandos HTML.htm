Comandos Iniciais do HTML

Tags Estruturais

Um documento okokookooHTML refere-se a um arquivo de texto salvo com a extensão .html ou .htm. Para ser interpretado corretamente pelo navegador, deve conter tags de marcação específicas.

Exemplo de Estrutura HTML

<!DOCTYPE html>
<html>
    <head>
        <title>Minha Página</title>
    </head>
    <body>
        Conteúdo da página
    </body>
</html>

Formatação de Texto

O HTML permite formatar textos utilizando diversas tags:

  • <b>: Texto em negrito
  • <i>: Texto em itálico
  • <u>: Texto sublinhado
  • <s>: Texto riscado

Parágrafos e Quebras de Linha

Os parágrafos são definidos com a tag <p> e as quebras de linha forçadas com <br>.

<p>Este é um parágrafo.</p>
<p>Outro parágrafo com uma quebra de linha.<br>Aqui começa uma nova linha.</p>

Títulos

Os títulos são definidos pelas tags <h1> a <h6>, sendo <h1> o mais importante e <h6> o menos importante.

<h1>Título de nível 1</h1>
<h2>Título de nível 2</h2>
<h3>Título de nível 3</h3>

Alteração de Fonte

A tag <font> permite alterar a fonte do texto (embora não seja recomendada no HTML5).

<font size="4" color="blue" face="Arial">Texto azul em Arial tamanho 4</font>
